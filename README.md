# Carrito de Seguimiento

## Descripción
Este proyecto consiste en el diseño y construcción de un carrito autónomo capaz de seguir una trayectoria mediante sensores. El sistema utiliza un microcontrolador para procesar la información de los sensores y controlar los motores del vehículo.

## Objetivo General
Diseñar e implementar un carrito autónomo capaz de seguir una línea utilizando sensores y un microcontrolador.

## Objetivos Específicos
- Diseñar el sistema electrónico del carrito.
- Implementar sensores de seguimiento.
- Programar el microcontrolador para controlar los motores.
- Integrar el hardware y el software del sistema.
